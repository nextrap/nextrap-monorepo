# @nextrap/style-button

## Zweck

Stellt die standardisierten Nextrap-Button-Klassen bereit.

**Für Buttons sollen in diesem Projekt diese Klassen verwendet werden.**
Keine eigenen abweichenden Button-Basisklassen einführen, wenn eine Variante aus `@nextrap/style-button` passt.

## Einbindung

```ts
import '@nextrap/style-button';
```

Oder als SCSS:

```scss
@use '@nextrap/style-button';
```

Die Styles basieren auf den globalen `--nt-*` Variablen aus `@nextrap/style-base`.

## Basisnutzung

Jeder Button basiert auf der Klasse:

- `.btn`

Typische Verwendung:

```html
<button class="btn btn-primary" type="button">Speichern</button>
<a class="btn btn-secondary" href="/zurueck">Zurück</a>
<input class="btn btn-success" type="submit" value="Absenden" />
```

## Verfügbare Button-Variationen

### Standard

- `.btn`
- `.btn-link`

### Farbvarianten

- `.btn-primary`
- `.btn-accent`
- `.btn-secondary`
- `.btn-success`
- `.btn-danger`
- `.btn-warning`
- `.btn-info`
- `.btn-light`
- `.btn-dark`

### Outline-Varianten

- `.btn-outline-primary`
- `.btn-outline-accent`
- `.btn-outline-secondary`
- `.btn-outline-success`
- `.btn-outline-danger`
- `.btn-outline-warning`
- `.btn-outline-info`
- `.btn-outline-light`
- `.btn-outline-dark`

### Größen

- `.btn-sm`
- `.btn-lg`

### Status / Verhalten

- `.disabled`
- `[disabled]`
- `.dropdown-toggle`

## Kombinierbare Beispiele

```html
<button class="btn btn-primary">Standard</button>
<button class="btn btn-outline-primary">Outline</button>
<button class="btn btn-success btn-sm">Klein</button>
<button class="btn btn-danger btn-lg">Groß</button>
<button class="btn btn-link">Link-Button</button>
<button class="btn btn-primary" disabled>Deaktiviert</button>
<button class="btn btn-secondary dropdown-toggle" type="button">Menü</button>
```

## Zusätzliche Hilfsklassen

### Button-Gruppen

Für zusammengehörige Buttons:

- `.btn-group`

Beispiel:

```html
<div class="btn-group">
  <button class="btn btn-primary">Links</button>
  <button class="btn btn-primary">Mitte</button>
  <button class="btn btn-primary">Rechts</button>
</div>
```

### Dropdown-Menü

Für das Menü passend zu `.dropdown-toggle`:

- `.dropdown-menu`

Beispiel:

```html
<button class="btn btn-primary dropdown-toggle" type="button">Aktionen</button>
<ul class="dropdown-menu">
  <li><a href="#">Bearbeiten</a></li>
  <li><a href="#">Löschen</a></li>
</ul>
```

## Agent-Hinweise

- Für Buttons immer mit `.btn` starten.
- Danach die passende Variante ergänzen, z. B. `.btn-primary` oder `.btn-outline-secondary`.
- Für deaktivierte Buttons das native `disabled`-Attribut bevorzugen.
- Für Button-Gruppen `.btn-group` statt eigener Layout-Klassen verwenden.
- Für Dropdown-Buttons `.dropdown-toggle` zusammen mit `.dropdown-menu` verwenden.
- Farbanpassungen immer in style-base.
- Wenn eine neue Button-Optik benötigt wird, zuerst prüfen, ob eine bestehende Variante ausreicht, bevor neue Klassen eingeführt werden.
