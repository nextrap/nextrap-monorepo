Parallax Background Component Demo

Scroll down to see the parallax effect in action. The background images move at a slower rate than the foreground content, creating a sense of depth.

Default: 100vh × 100vw

Mountain Landscape

This first example uses the default settings - 100vh height and 100vw width. The parallax effect creates a beautiful depth illusion as you scroll.

Custom Height: 60vh

Forest Scene

This example demonstrates a custom height of 60vh, perfect for creating smaller parallax sections within your page layout.

With Background Color

Night Sky

Here we've added a dark background color (#1a1a2e) that shows through if the image fails to load or during loading, providing a better user experience.

Custom: 50vh × 80%

Flexible Sizing

This example shows custom width (80%) and height (50vh), demonstrating how the component can be sized to fit your design needs. Notice how it's centered within the content section.

Full Viewport Finale

End of Demo

That's it! The parallax background component is simple to use yet powerful. Just set the image attribute and optionally customize the height, width, and background color.
