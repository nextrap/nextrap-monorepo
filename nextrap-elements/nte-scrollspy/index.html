NTE Scrollspy Component - Examples

Basic Information & Options

The scrollspy component can be configured with:

              
    Custom CSS classes to apply to the element
    classes?: string[];
  
    Custom attributes to apply to the element
    attributes?: {
      dataTarget?: string;
    };
  
    Orientation of the scrollspy element, defaults to 'vertical'
    orientation?: ScrollspyOrientation;
  
    Navigation items to display in the scrollspy
    data?: INavigationItem[id, label, href, active];
  
    Track progress through sections
    trackProgress?: boolean;

    Whether to create links for the navigation items
    linkItems?: boolean;

Basic example

Settings:

  • orientation: vertical
  • trackProgress: false
  • Provide custom data
              
  data: [ 
      { id: 'section-1-1', label: 'First Section'}, 
      { id: 'section-2-2', label: 'Second Section' }, 
      { id: 'section-1-3', label: 'Third Section' }, 
      { id: 'section-1-4', label: 'Fourth Section' },
    ]
