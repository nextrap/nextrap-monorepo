# @nextrap/nte-nav

Navigation web components: `nte-nav`, `nte-navbar`, `nte-navbar-line`.

## Import

```ts
import '@nextrap/nte-nav';
```

For theme/demo styling:

```scss
@use '@nextrap/nte-nav' as nav;

nte-nav.style-default {
  @include nav.default-style();
}

nte-navbar.style-default {
  @include nav.navbar-default-style();
}

nte-navbar-line.style-default {
  @include nav.navbar-line-default-style();
}
```

`nte-nav` gets `style-default` automatically from `nte-core` when no `style-*` class exists. Do not add visual styles to bare `nte-nav` selectors. Use `nav-horizontal` / `nav-vertical` for orientation and `with-*` classes for modifiers, e.g. `with-undecorated`, `with-pills`, `with-primary`.

```html
        </li>

<nte-offcanvas id="navSidebar">
  <nte-nav class="nav-vertical"></nte-nav>
</nte-offcanvas>
```

## Parts

`nte-nav`: `nav`, `burger-wrapper`, `burger-default`, `menutext`, `burger`, `main`.
`nte-navbar`: `wrapper`, `spacer`, `navbar`, `brand`.
`nte-navbar-line`: `main`, `container`, `brand`, `nav`.
