ARIA Support Keyboard Nav Screen Readers Focus Management

ARIA Support

Full ARIA implementation including:

  • role="tablist" on container
  • role="tab" on each tab
  • role="tabpanel" on panels
  • aria-selected state management
  • aria-controls and aria-labelledby relationships

Keyboard Navigation

  • Tab: Move focus into and out of the tab list
  • Arrow Right/Down: Move to next tab
  • Arrow Left/Up: Move to previous tab
  • Home: Move to first tab
  • End: Move to last tab
  • Space/Enter: Activate focused tab

Screen Reader Support

Properly announces tab states and relationships to assistive technologies.

Each tab has a label attribute for screen reader context.

Focus Management

Smart focus management with roving tabindex pattern.

Only the active tab is in the tab order (tabindex="0").
