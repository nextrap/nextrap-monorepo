nte-theme-switcher

A lightweight Web Component for switching themes via data-theme attribute. Lit

Live Demo

The theme switcher is active in the bottom-left corner of this page. Select a theme to see the data-theme attribute change on <html>.

Current data-theme: Check <html> element in DevTools to see the applied attribute.

Usage

<nte-theme-switcher
  themes="light, dark, high-contrast"
  target="html"
></nte-theme-switcher>

Properties

Property Type Default Description
themes string "" Comma-separated list of theme names
target string "html" Target element: "html", "body", "current", or an element ID

Features

  • URL Sync — Theme persists via ?theme=... URL parameter
  • Flexible Target — Set data-theme on html, body, the component itself, or any element by ID
  • Default Option — "Default / System" removes the attribute and cleans the URL
  • No Dependencies — Only requires Lit
  • Shadow DOM — Styles are encapsulated

Target Options

Value Behavior
"html" Sets data-theme on document.documentElement
"body" Sets data-theme on document.body
"current" Sets data-theme on the <nte-theme-switcher> element itself
Any other string Interpreted as element ID via document.getElementById()
