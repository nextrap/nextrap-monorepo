nte-theme-switcher

A lightweight Web Component for switching themes via CSS classes on the <body> element. Lit

Live Demo

The theme switcher is active below. Select a theme to see the CSS class change on <body> (e.g. class="theme-dark").

Current theme class: Check <body> element in DevTools to see the applied class (e.g. theme-light, theme-dark).

Usage

<nte-theme-switcher
  themes="light, dark, high-contrast"
></nte-theme-switcher>

Properties

Property Type Default Description
themes string "" Comma-separated list of theme names (classes will be prefixed with theme-)

Features

  • CSS Class Based — Sets theme-{name} class on <body>
  • URL Sync — Theme persists via ?theme=... URL parameter
  • Default Option — "Default / System" removes the class and cleans the URL
  • No Dependencies — Only requires Lit
  • Shadow DOM — Styles are encapsulated

Styling with Theme Classes

Use the theme classes in your CSS to style elements based on the selected theme:

/* Default styles */
.my-element { background: white; color: black; }

/* Dark theme */
.theme-dark .my-element { background: #1a1a1a; color: white; }

/* Light theme */
.theme-light .my-element { background: #fafafa; color: #333; }
