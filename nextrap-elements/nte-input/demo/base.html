nte-input

Concise demo for input primitives and styling options.

Usage
- `nte-input` renders its own native control; don’t provide an `<input>`/`<select>` child.
- Inline layout is default; add class `multiline` for stacked label layout.
- Use `--nxa-label-cols` / `--nxa-input-cols` with `var(--cols-X)` (12-col grid) to tune widths.
- Use classes `size-sm` / `size-lg` and `style-modern` for sizing and modern styling.

Combinations

Default (inline)

Label left, input right (default label 4/12, input 6/12).

Multiline

Add class `multiline` to stack label above the field.

Floating Label

Uses the `floating` property (layout/cols don’t apply).

Cols + Size (1/3)

2/10 split + `size-sm`.

Cols + Size (2/3)

4/8 split + default size.

Cols + Size (3/3)

6/6 split + `size-lg`.

Modern Style

Add class `style-modern`.

Label Fallback

If no label is provided, the control `name` becomes the label (placeholder is ignored).

No Label

No label is shown when neither label nor name is provided.

Native Input

`nte-input-control` can also wrap native controls directly.
