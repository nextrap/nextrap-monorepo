HTML Demo Support

This is an HTML demo file loaded inside an iframe by nte-demo-viewer.

Key difference: HTML demos run in an isolated iframe, so scripts execute normally. Markdown demos are rendered inline via tj-markdown-loader.

How It Works

  • Files ending in .html or .htm are loaded in an <iframe>
  • The iframe auto-resizes to match the content height
  • Content changes are observed via MutationObserver
  • Code editing is disabled for HTML demos (the </> button is hidden)
Note: Cross-origin iframes cannot be auto-resized. In that case, a fallback height of 600px is used.

Usage

<demo
  title="HTML Demo"
  description="Interactive HTML content"
  src="/demo/my-demo.html"
></demo>
