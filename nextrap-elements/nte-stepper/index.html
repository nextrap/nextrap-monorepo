NTE Stepper

A comprehensive, customizable stepper component with multiple modes, navigation options, and styling capabilities

Component Overview

The NTE Stepper component provides a flexible way to guide users through multi-step processes with various display modes and navigation options.

Property Type Default Description
mode 'horizontal' | 'vertical' | 'circular' 'horizontal' Display mode of the stepper
linear boolean false Restricts navigation to sequential steps only
href string undefined URL or anchor link for navigation (on individual steps)
status 'inactive' | 'active' | 'disabled' 'inactive' Current status of the step
